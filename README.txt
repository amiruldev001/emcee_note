
EMCEE FLASH SLIDES – PWA PACK
=============================
How to use:
1) Unzip this pack to any folder or a simple web host (e.g., GitHub Pages, Netlify, or a local HTTP server).
2) Open index.html in your mobile browser.
3) Tap the ••• browser menu and choose "Install app" / "Add to Home Screen".
4) Use tap (right=next, left=back), swipe, or arrow keys to navigate.

Files included:
- index.html                (the slide app)
- manifest.webmanifest      (PWA manifest)
- sw.js                     (service worker for offline)
- /icons/*.png              (app icons)

Notes:
- Dark mode is default; tap the Dark/Light pill to toggle.
- The app adapts to portrait/landscape; a hint appears when portrait.
- Offline works after first load while connected.
